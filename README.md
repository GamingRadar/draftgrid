# DraftGrid

DraftGrid is a lightweight web tool for generating clean, printable drawing guides for designers, artists, and product thinkers.

It helps you create structured 2D and 3D sketch layouts instantly, ready to export as high-quality A4 PDFs.

## 🎯 Why DraftGrid?

While taking a concept art course, I noticed how much time was spent manually creating layout guides.

DraftGrid solves that by letting you generate:
- Structured grids
- 3D sketch guides
- Professional design sheets

...in seconds.

---

## 🚀 Features

### 📐 2D Guides
- Adjustable grid (2–12 boxes)
- Aspect ratios (1:1, 2:3, 3:4, custom)
- Margins & spacing control
- Portrait / Landscape A4

### 🧊 3D Guides
- Basic shapes (cube, cuboid, cylinder, etc.)
- Isometric / perspective layouts
- Depth & angle control
- Ideal for product sketching

### 🧾 Professional Sheet Layout
- Title, version, author
- Date & page numbering
- Multi-page export
- Optional logo support

### 🎨 Customization
- Stroke color & weight
- Dotted / solid guides
- Layout presets
- Light / dark UI

### 📤 Export
- High-resolution PDF (print-ready)
- Clean, centered layouts
